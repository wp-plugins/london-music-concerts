== Description ==

The 'London Music Concerts' plugin will display a constantly updating list of concerts and gigs taking place in the capital. All of the big venues are included: O2 Arena, O2 Empire, Wembley Arena, Roundhouse, HMV Forum, Hammersmith Apollo, Royal Albert Hall, Royal Festival Hall, and many more.

The listing will include a title, description and date for each gig, alongside an image of the band/artist. There are options to amend the date format, number of items displayed, and all of the CSS styles.

It has also been widgetized for drag-and-drop use on your sidebar.

== Installation ==

1. Upload the `london-music-concerts` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu
3. Navigate to 'Settings > London Music Concerts' to configure the plugin
4. You can either drag the `London Music Concerts` widget straight onto your sidebar, or you can place `<?php london_music_concerts(); ?>` into your template where you'd like the events to appear

== Upgrade ==

1. Delete the old `london-music-concerts` folder
2. Upload the new `london-music-concerts` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu
4. Navigate to 'Settings > London Music Concerts' to configure the plugin

== Frequently Asked Questions ==

= Where are the events taken from? =

The events are continually updated at the London Drum website -- http://www.londondrum.com/events/?cat=856

= How do I uninstall the plugin? =

Simply delete the old `london-music-concerts` folder, and remove the `<?php london_music_concerts(); ?>` line from your template file.

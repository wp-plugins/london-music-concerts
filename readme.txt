Displays a list of upcoming comedy gigs in London. All of the big name comedians are included, plus all the little stand-up shows and West End theatre runs.

== Description ==

The 'London Comedy Gigs' plugin will display a constantly updating list of comedy gigs in the capital, with all the big name comedians, little stand-up shows and West End theatre runs.

The listing will include a title, description and date for each gig, alongside a picture of the comedian. There are options to amend the date format, number of games displayed, and all of the CSS styles.

It has also been widgetized for drag-and-drop use on your sidebar.

== Installation ==

1. Upload the `london-comedy-gigs` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu
3. Navigate to 'Settings > London Comedy Gigs' to configure the plugin
4. You can either drag the `London Comedy Gigs` widget straight onto your sidebar, or you can place `<?php london_comedy_gigs(); ?>` into your template where you'd like the events to appear

== Upgrade ==

1. Delete the old `london-comedy-gigs` folder
2. Upload the new `london-comedy-gigs` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu
4. Navigate to 'Settings > London Comedy Gigs' to configure the plugin

== Frequently Asked Questions ==

= Where are the comedy gigs taken from? =

The gigs are continually updated at the London Drum website -- http://www.londondrum.com/events/?cat=9

= Is there a plugin for other types of event? =

We have four more plugins devoted to events in London -- 
'London Events Guide' -- http://wordpress.org/extend/plugins/london-events-guide/
'London Music Concerts' -- http://wordpress.org/extend/plugins/london-music-concerts/
'London Theatre Guide' -- http://wordpress.org/extend/plugins/london-theatre-guide/

And an application for Facebook users -- http://apps.facebook.com/london-events-guide/

= How do I uninstall the plugin? =

Simply delete the old `london-comedy-gigs` folder, and remove the `<?php london_comedy_gigs(); ?>` line from your template file.
